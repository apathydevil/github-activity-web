# flask-intro

## Description
A simple Flask application that allows users to enter a GitHub username and view their recent activity. The app fetches data from the GitHub API and displays it in a user-friendly format. It is a Web application version of the command-line tool I built beforehand.

## Usage
1. `git clone [https://github.com/apathydevil/github-activity-web](https://github.com/apathydevil/github-activity-web)`
2. `cd github-activity-web`
3. `pip install -r requirements.txt`
4. `python main.py`
5. Open your browser and navigate to `http://127.0.0.1:5000`
